## 🚀 Dev Workflow - Smooth & Steady 🎯

### **Branching 101**
- Work in **dev** like a pro:
  ```bash
  git checkout dev
  ```
- Push to **dev**, merge into **main** after review.

### **Coding Rituals**
```bash
# Get latest updates
git pull origin dev

# Work your magic
git add .
git commit -m "🚀 Added feature X"

git push origin dev

# Merge to master
git checkout master
git merge dev
git push origin master
```

### **Deploying Like a Boss**
- **Auto magic:** Netlify deploys from `master`.
- **Manual flex:**
  ```bash
  netlify login
  netlify deploy --prod
  ```
